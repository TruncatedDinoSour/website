Fixing kernel freezing on realtek WiFi

2022-01-20 16:11:48 EET | back | home | git

Internet way

  • Run

sudo curl 'https://ari-web.xyz/data/fix_rtw.conf' -o /etc/modprobe.d/fix_rtw.conf

  • Reboot

sudo reboot -h

No-internet way

  • Open /etc/modprobe.d/fix_rtw.conf (you can name it whatever you want) in any editor as root
  • Add this to the file:

options rtw88_pci disable_aspm=1

  • Reboot

sudo reboot -h
